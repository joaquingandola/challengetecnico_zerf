# challengetecnico_zerf

Developer - File System

El objetivo es diseñar, de manera orientada a objetos, las clases y relaciones necesarias para modelar un file system similar al de Linux. 

Comandos:
    ● cd(dirName)
    ● touch(fileName)
    ● ls()
    ● mkdir(dirName)
    ● pwd()
    ● rm(dirName / fileName) -> Es el único comando que funciona por rutas, para el resto no tuve el tiempo.
